About Reveel
Reveel is a full-stack web application that enables users to generate stunning images using AI prompts, share their creations, and explore images generated by others. With a sleek UI and powerful backend, Reveel brings creativity and community together in one seamless platform.

Features
AI Image Generation: Create unique images based on text prompts.

Post Creations: Share generated images on your profile.

Explore & Search: Discover images created by other users with a powerful search feature.

Responsive UI: Clean and intuitive interface built with React and Material-UI.

Full-stack Implementation: Backend and frontend fully integrated for smooth user experience.
